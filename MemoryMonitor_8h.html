Memory monitoring utilities for AVR microcontrollers.

Author

Provides functions to measure free RAM between heap and stack. Essential for debugging memory issues on resource-constrained systems.

Function Documentation

◆ getFreeMemory()

int getFreeMemory ( )
inline

Calculates free RAM between heap and stack.

Returns
Free memory in bytes

Uses AVR memory layout to calculate space between heap top (__brkval) and stack pointer (SP). Returns difference between stack pointer and either __brkval or __heap_start.

Note
Result may vary during execution as stack grows/shrinks
