Detailed Description

Low-level sensor abstractions and hardware drivers.

Author
Andrea Bortolotti
Version
2.0

Provides template-based sensor interface and concrete implementations for various sensor types:

  • LM75 I2C temperature sensor
  • Analog photoresistor
  • HC-SR501 PIR motion sensor
  • Virtual sensors (RAM, VCC, Loop Time)
Note
This file must remain header-only due to template classes.
