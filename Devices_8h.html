Detailed Description

Device class declarations for lights, sensors, and automation.

Author
Andrea Bortolotti
Version
2.0

Provides all device abstractions for the smart home system:

  • Light devices (Simple, Dimmable, RGB, Outside)
  • Sensor devices (Temperature, Light, PIR, RAM, VCC, LoopTime)
  • Device factory for convenient instantiation
