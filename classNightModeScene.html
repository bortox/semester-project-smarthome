Detailed Description

Low priority scene that limits global brightness.

Priority: 10 (Low - applied first, can be overridden by others)

Reduces all light brightness to 20% by modifying the global multiplier. This is a passive effect applied via DimmableLight::setBrightnessMultiplier(), so update() is empty.
