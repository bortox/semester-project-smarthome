Detailed Description

template<typename T>
class DynamicArray< T >

Memory-optimized dynamic array implementation.

Template Parameters
TType of elements stored in the array

A lightweight vector-like container optimized for embedded systems.

Warning
Not interrupt-safe.
Uses manual memory management (new/delete). Ensure heap space is sufficient.

Growth Strategy:

  • Initial capacity: 0
  • Growth step: 4 elements
  • Max capacity: 64 elements
  • Shrink threshold: 8 unused slots
