Detailed Description

template<typename T>
class DynamicArray< T >

Memory-efficient dynamic array for embedded systems.

Template Parameters
TElement type (typically pointers)

Template-based container optimized for AVR microcontrollers. Grows dynamically as needed with configurable growth factor. Must remain header-only due to template instantiation requirements.

Note
Memory is allocated from heap. Monitor free RAM when using.
