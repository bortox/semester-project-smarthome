Detailed Description

Core system infrastructure for event-driven embedded architecture.

Author
Andrea Bortolotti
Version
2.0

Provides the foundational components for the smart home system:

  • Dynamic array container (template-based, header-only)
  • Event system for decoupled communication
  • Device registry for centralized device management
  • Base device interface
Note
This file must remain header-only due to template classes.
