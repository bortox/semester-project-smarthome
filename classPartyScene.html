Detailed Description

Medium priority scene that cycles RGB light colors.

Priority: 50 (Medium - can be overridden by Alarm, overrides Night Mode)

Cycles all RGBLight devices through Red -> Green -> Blue -> repeat every 500ms using non-blocking timing.
